for문은 C언어에 있어 필수적인 요소인 반복문의 일부이며 js, python 등의 다양한 언어들에서도 비슷한 형태로 응용되기에 **굉장히 상당히 매우 중요하다.**

# for문 어려워! 🤯
보통 for문을 어렵다고 하는 경우는 다음과 같다.  
>1. 나는 while문이 익숙한데 하나의 **인수가 3개나 되니 어렵다.**
>2. **뭔 개소린지 모르겠다.**

2번은 보통 **기초가 부족하기 때문에** 발생한다. 이런 경우는 기초 100제를 배껴서 제출한 친구들이 해당한다.  
이런 친구들은 **제발 기초 100제는 풀어줬으면 좋겠다.**  
이건 C언어 공부와 알고리즘을 공부하기 위한 정말 '최소한'의 노력이다. 이 정도도 하지 않은 사람을 이해시켜줄 자신도 없고 마음도 없다.

대부분의 친구들은 1번에 해당한다. 1번의 경우는 보통 **for문의 정의**를 정확히 알지 못해 발생한다.  
기초가 탄탄한다면 for문 정도 정의만 정확하게 인지한다면 빠르게 마스터 할 수 있다.

# for문을 왜 사용하는데?? 🤔
앞서 말했든 while문에 익숙한 친구들이 for문의 필요성을 느끼지 못하는 경우가 많다.  
단언해서 말하건데 **while문과 for문은 사용해야 하는 상황이 명확하게 구분되어있다.**  

다음은 1부터 100까지의 수를 출력하는걸 while문을 통해 나타낸 것이다.
```c
int i=1
while(i<=100) {
    printf("%d ", i);
    i++;
}
```
위 코드는 **굉장히 비효율적이다.** 반복문에서 한번 밖에 안쓸 변수, 조건문, 그리고 변수에 1을 더하는 것에 각각 한 줄 씩이나 부여해주고 있다.  
만약 우리가 이런 반복문을 3개만 만든다고 해도 5x3=15줄이나 되는 코드를 허비해야한다.  


이 5줄짜리 비효율적인 코드를 for문으로 2줄컷 낼수 있다.  
```c
for(int i=1; i<=100; i++)
    printf("%d ", i);
```
for문은 위에서 볼 수 있듯 변수 선언, 조건문, 증감 등 반복문에서 자주 사용하는 요소를 하나로 묶어 일정한 횟수의 반복을 쉽게 처리할 수 있다.

**즉, 횟수가 정해져있는 반복문에서는 for문이 훨씬 유리하다.**  
(다만, 횟수가 정해져 있지 않고 조건이 정해진 반복문은 while문을 사용하는게 유리함에 유의하자.)

# for문이 뭔데? 🎒
for문을 크게 다음 세가지의 요소로 구성되어 있다.
![](/images/for.png)
위의 코드에서 보았듯 for 문은 **초기식, 조건식, 증감식** 3가지로 구성되어 있다. 각각에 대해 자세히 알아보자.  

## 1. 초기식
초기식은 **반복문에 사용할 변수를 초기화 해준다.**
경우에 따라 전역으로 선언된 변수를 초기화하여 사용할 수도, 또는 지역 변수로 새로 선언 할 수도 있다.  
반복문 밖에서 다시 반복문을 활용해야 하는 경우에는 **전역 선언**을, 밖에서 활용하지 않는 경우에는 **지역 변수**로 선언해주자.  
대부분의 경우 지역변수로 선언하는 것이 메모리를 아끼는 것에 도움이 된다.

## 2. 조건식
조건식은 **반복을 할지 판단해준다.**
while문 내부의 조건문과 완전히 동일한 역할이다. 반복한 단계를 끝나고 돌아왔을때 다시 실행할지 판단한다.

## 3. 증감식
증감식은 **반복문이 끝날때 마다 실행된다.**
보통 `i--`또는 `i++`와 같은 간단한 말 그대로의 '증감식'이 사용되지만 필요에 따라 **함수**가 사용될 수도 있다.

# for문의 활용 💻
for문은 생각보다 굉장히 많은 곳에서 활용될 수 있다. 따라서 <s>맞으면서</s> 직접 따라하면서 배우는게 가장 빠르다.  
다음 예제들을 **직접 타이핑**하고 실행해봄으로서 for문과 친해져보자.

## 1. 1부터 100까지
```c
for(int i=1; i<=100; i++)
    printf("%d ", i);
```
위에서 보았던 코드니 추가적인 설명은 하지 않겠다.

## 2. 0부터 9까지
```c
int l[10]={};
for(int i=0; i<10; i++)
    l[i]=3*i;
for(int i=0; i<10; i++)
    printf("%d ", i);
```
배열을 알다싶이 `size-1`칸까지 만들어 진다. 따라서 **0에서 size 미만 까지의 반복을 통해 배열을 처리하는 경우가 굉장히 많다.**  
이 반복문은 외워두는 것을 강력하게 추천한다.

## 3. a부터 b까지 모두 더하기
```c
int a, b, sum=0;
scanf("%d %d", &a, &b);
for(int i=a; i<=b; i++)
    sum+=i;
printf("%d", sum);
```
`a`부터 `b`까지 `i`를 차례로 증가시키며 `sum`에 더한 후 출력하는 코드이다.  

## 4. 구구단 (3단)
```c
for(int i=1; i<=9; i++)
    printf("3 x %d = %2d\n", i, 3*i);
```
`i`를 증가시키며 구구단중 3단을 출력하는 코드다.

## 5. 구구단 (1~9단)
```c
for(int i=1; i<=9; i++){
    for(int j=1; j<=9; j++)
        printf("%d x %d = %2d\n", i, j, i*j);
}
```
1단부터 9단까지의 구구단을 출력하는 코드이다.  
`i`를 증가시키며 각 단을 증가시키고, 각 단에서는 `j`를 증가 시키면서 그 단에 해당하는 구구단을 출력한다.  
**2중 for문의 가장 기본적인 형태이는 정확하게 이해하고 넘어가자.**

## 심화 : 엔터를 입력할 때 까지 입력받고 출력하기
```c
for(char s[100]=''; scanf("%s", s)!=EOF; printf("%s", s));
```
우리가 굉장히 많은 줄을 써서 만들었던 입력받은걸 그대로 반환하는 코드를 for문으로는 한줄컷 낼 수 있다.

이 코드는 그냥 조금의 영감을 주기 위해 넣은 코드로 이런 코드를 짜라는 미친 소리는 아니다. 사실 숏코딩 하기는 좋겠지만 '안전한' 코드는 아니다.  
앞서 잠깐 언급하긴 했지만 **증감식이 꼭 '증감식'일 필요는 없다.**   
코딩에서 가장 중요한건 나의 습관을 버리고 정의에 따라 창의력을 발휘하는 것이란걸 알려주려고 넣었다.
